نصب TunnelMod

راهنمای نصب نسخه عمومی پنل مدیریت تونل روی Ubuntu 20.04، 22.04 و 24.04

۱پیش‌نیازها

  • سرور ورودی Ubuntu با دسترسی کامل sudo
  • سرور مقصد با دسترسی root برای راه‌اندازی خودکار WireGuard
  • باز بودن پورت 8443/TCP روی سرور ورودی

۲دریافت از GitHub

ssh USERNAME@YOUR_SERVER_IP
sudo apt update && sudo apt install -y git
git clone https://github.com/hazhan4268/tunnelmod.git
cd tunnelmod
chmod +x install.sh uninstall.sh scripts/tunnel-panel-helper

۳نصب

sudo ./install.sh

IP عمومی سرور را تأیید و یک رمز مدیر با حداقل ۱۰ کاراکتر تعیین کنید.

برای نصب پنل، ورود مستقیم root لازم نیست؛ داشتن دسترسی sudo کافی است.

۴ورود

https://YOUR_SERVER_IP:8443
گواهی اولیه Self-Signed است. اثر انگشت نمایش‌داده‌شده در پایان نصب را بررسی کنید.

۵افزودن مقصد

در بخش «سرورها»، IPv4، پورت SSH، کاربر root و رمز موقت سرور مقصد را وارد کنید. رمز فقط برای نصب کلید SSH استفاده می‌شود و ذخیره نخواهد شد.

۶ساخت تونل

  • روش پیشنهادی: WireGuard + DNAT
  • پروتکل: TCP، UDP یا هردو
  • پورت ورودی و مقصد: برای HTTPS مقدار 443

HAProxy فقط TCP را پشتیبانی می‌کند.

۷عیب‌یابی

sudo systemctl status tunnel-panel --no-pager
sudo journalctl -u tunnel-panel -n 100 --no-pager
sudo wg show
sudo iptables -t nat -L -n -v
