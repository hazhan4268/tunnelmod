نصب TunnelMod

راهنمای نصب کوتاه، به‌روزرسانی و SSL دامنه

۱پیش‌نیازها

  • Ubuntu 20.04 یا جدیدتر
  • دسترسی کامل sudo
  • باز بودن پورت 8443/TCP
  • برای SSL دامنه: DNS صحیح و پورت 80/TCP باز

۲نصب سریع

wget -qO install-online.sh https://raw.githubusercontent.com/hazhan4268/tunnelmod/main/install-online.sh
sudo bash install-online.sh

۳نصب کلاسیک

git clone https://github.com/hazhan4268/tunnelmod.git && cd tunnelmod
sudo bash install.sh

۴ورود به پنل

https://YOUR_SERVER_IP:8443
گواهی اولیه Self-Signed است؛ اثر انگشت نمایش‌داده‌شده در پایان نصب را بررسی کنید.

۵SSL دامنه

sudo tunnelmod-domain panel.example.com admin@example.com

دامنه باید با رکورد A به IP سرور وصل باشد. آدرس پنل پس از فعال‌سازی: https://panel.example.com:8443

۶عیب‌یابی

sudo tunnelmod-diagnose

خروجی این ابزار IPها را خودکار مخفی می‌کند.

۷به‌روزرسانی

sudo tunnelmod-update

پیش از به‌روزرسانی Backup ساخته می‌شود و در صورت خطا نسخه قبلی خودکار برمی‌گردد.
